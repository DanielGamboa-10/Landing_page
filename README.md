# 15 Interfaces Enlazadas - Práctica de Diseño con HTML y CSS

## 📌 Descripción del Proyecto

Este proyecto consiste en el desarrollo de 55 páginas web estáticas enlazadas entre sí, creadas únicamente con HTML5 y CSS3.

El objetivo principal es fortalecer las habilidades de maquetación y diseño visual, sin utilizar JavaScript ni funcionalidades interactivas. Cada página representa una pequeña interfaz enfocada en estructura, estilo y organización visual.

---

## 🎯 Objetivos

- Practicar estructura semántica en HTML.
- Mejorar el manejo de CSS.
- Desarrollar criterio de diseño visual.
- Trabajar la jerarquía visual y organización del contenido.
- Aumentar velocidad y consistencia en el desarrollo de interfaces.
- Comprender la navegación básica entre múltiples páginas.

---

## 🛠 Tecnologías Utilizadas

- HTML5
- CSS3
- Flexbox
- Principios básicos de diseño responsivo
- Enlaces internos entre páginas

---

## 📂 Estructura del Proyecto

El proyecto incluye:

- 55 archivos HTML.
- Un sistema de estilos en CSS.
- Navegación básica entre páginas.
- Variaciones en diseño, distribución y estilos visuales.

Cada interfaz explora diferentes conceptos como:

- Secciones tipo landing page.
- Layouts con hero section.
- Sistemas de tarjetas (cards).
- Barras de navegación.
- Distribuciones en columnas.
- Diseños minimalistas.
- Variaciones de tipografía y color.

---

## 📈 Enfoque de Aprendizaje

Este proyecto está enfocado en la repetición estratégica como método de mejora.  
Al crear múltiples interfaces, se fortalece:

✔ La organización del código.  
✔ La consistencia visual.  
✔ El manejo de márgenes y espacios.  
✔ La combinación de colores.  
✔ La estructura limpia y ordenada del CSS.
